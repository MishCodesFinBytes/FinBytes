The Debt Snowball Method:

Before we dive into the code, let's briefly understand the debt snowball method. It is a popular debt repayment strategy where the focus is on paying off the smallest debt first while making minimum payments on other debts. Once the smallest debt is paid off, the amount that was being used for its repayment is then directed towards the next smallest debt, creating a "snowball" effect. This method helps in building momentum and motivation as debts are paid off one by one, making it easier to tackle larger debts in the future.

Step 1: Importing the libraries and dataset


#Importing libraries
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from tkinter import *
from tkinter import messagebox

#Loading the dataset
df = pd.read_csv('debt_data.csv')

Step 2: Creating a function to calculate the total debt balance


#Function to calculate total debt balance
def calculate_total_debt():
    total_debt = df['Balance'].sum()
    return total_debt

Step 3: Creating a function to calculate the minimum monthly payment


#Function to calculate minimum monthly payment
def calculate_min_payment():
    min_payment = df['Min Payment'].sum()
    return min_payment

Step 4: Creating a function to calculate the interest paid per month


#Function to calculate interest paid per month
def calculate_interest_paid():
    interest_paid = (df['Balance'] * (df['Interest Rate']/100))/12
    return interest_paid.sum()

Step 5: Creating a function to calculate the debt payoff timeline


#Function to calculate debt payoff timeline
def calculate_payoff_timeline():
    df['Months to Payoff'] = np.ceil(df['Balance']/df['Min Payment'])
    return df['Months to Payoff'].max()

Step 6: Creating a function to update the debt balance after each payment


#Function to update debt balance after each payment
def update_debt_balance():
    df['Balance'] = df['Balance'] - df['Min Payment']
    return df['Balance']

Step 7: Creating a function to implement the debt snowball method


#Function to implement the debt snowball method
def debt_snowball():
    df['Rank'] = df['Balance'].rank(method='first')
    df.sort_values(by=['Balance'], inplace=True)
    df['Debt Snowball Payment'] = df['Min Payment'] + (df['Rank']-1)*100
    return df['Debt Snowball Payment']
