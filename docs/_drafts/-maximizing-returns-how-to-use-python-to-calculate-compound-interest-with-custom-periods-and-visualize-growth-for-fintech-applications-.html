Understanding Compound Interest:

Compound interest is the interest earned on the initial principal amount as well as the accumulated interest from previous periods. In simple terms, it is the interest on interest. This concept is widely used in the finance industry to calculate the returns on investments and the cost of borrowing money. The formula for calculating compound interest is A = P(1+r/n)^nt, where A is the final amount, P is the principal amount, r is the annual interest rate, n is the number of compounding periods per year, and t is the total number of years.

Generating a Python Script:

To generate a Python script for calculating compound interest, we first need to define a function that takes in the necessary parameters and returns the final amount. Let's call this function `compound_interest()`.

```python
def compound_interest(principal, rate, periods, years):
final_amount = principal * (1 + (rate/periods))**(periods*years)
return final_amount
```
Next, we can call the function and pass in the required values to calculate the compound interest. For example, let's assume we want to calculate the compound interest for a principal amount of $10,000 with an annual interest rate of 5% for 5 years, compounded semi-annually. To do so, we can call the function as follows:

```python
final_amount = compound_interest(10000, 0.05, 2, 5)
print(final_amount)
```

This would give us a final amount of $12,762.82. However, this script only calculates the final amount and does not give us a visual representation of the growth over time. To visualize the growth, we can use Python's data visualization library, Matplotlib.

Visualizing Growth with Charts:

Matplotlib is a powerful library that allows us to create various types of charts and graphs. In our case, we will use the `pyplot` module of Matplotlib to plot a line chart showing the growth of the investment over the 5-year period.

First, we need to import the necessary modules and define the parameters for our chart. We will use the same values as before, with the addition of a variable called `time` that will hold a list of years from 1 to 5.

```python
import matplotlib.pyplot as plt
%matplotlib inline

principal = 10000
rate = 0.05
periods = 2
years = 5
time = list(range(1,6))
```

Next, we can use the `compound_interest()` function to calculate the final amount for each year and store it in a list called `amounts`.

```python
amounts = []
for year in time:
final_amount = compound_interest(principal, rate, periods, year)
amounts.append(final_amount)
```

Now, we are ready to plot the line chart using the `plt.plot()` function. We will pass in the `time` variable as the x-axis and the `amounts` variable as the y-axis.

```python
plt.plot(time, amounts)
plt.title("Growth of Investment Over Time")
plt.xlabel("Years")
plt.ylabel("Final Amount ($)")
plt.show()
```

This will result in a line chart that shows the growth of the investment over the 5-year period. We can see that the growth is not linear, and the final amount increases significantly in the last few years due to the effect of compounding.
