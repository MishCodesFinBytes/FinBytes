Introduction:

In today's fast-paced financial world, it is essential to have access to real-time data and information on popular indices such as the S&P 500, NASDAQ, and FTSE. These indices act as indicators of the overall performance of the stock market and are closely monitored by investors and financial analysts to make informed decisions. With the advancements in technology and the rise of fintech, there is an increasing demand for efficient and user-friendly tools that allow users to track the performance of these indices and analyze historical trends. In this blog post, we will explore how to develop a tracker for popular indices using Python, which will enable users to monitor the performance and view historical trends of these indices.

Concept Explanation:

To develop a tracker for popular indices, we will be using the Python programming language and its powerful libraries such as Pandas, Matplotlib, and Requests. The tracker will fetch real-time data from reliable sources and display it in a user-friendly format, allowing users to monitor the performance of popular indices.

Step 1: Importing Libraries

The first step is to import the necessary libraries for our project. We will be using Pandas for data manipulation, Matplotlib for data visualization, and Requests for fetching data from the web.


import pandas as pd
import matplotlib.pyplot as plt
import requests

Step 2: Fetching Data

Next, we need to fetch the data for the popular indices - S&P 500, NASDAQ, and FTSE. We will use the Requests library to make HTTP requests and fetch the data in JSON format.


# Fetching data for S&P 500
sp500_url = 'https://financialmodelingprep.com/api/v3/quotes/index?apikey=demo&symbol=^GSPC'
sp500_data = requests.get(sp500_url).json()

# Fetching data for NASDAQ
nasdaq_url = 'https://financialmodelingprep.com/api/v3/quotes/index?apikey=demo&symbol=^IXIC'
nasdaq_data = requests.get(nasdaq_url).json()

# Fetching data for FTSE
ftse_url = 'https://financialmodelingprep.com/api/v3/quotes/index?apikey=demo&symbol=^FTSE'
ftse_data = requests.get(ftse_url).json()

Step 3: Organizing Data

Once we have fetched the data, we need to organize it in a structured format for further analysis and visualization. We will create a dataframe using the Pandas library, which will contain all the necessary information for our tracker.


# Creating dataframes for each index
sp500_df = pd.DataFrame(sp500_data)
nasdaq_df = pd.DataFrame(nasdaq_data)
ftse_df = pd.DataFrame(ftse_data)

# Combining all dataframes into one
indices_df = pd.concat([sp500_df, nasdaq_df, ftse_df], ignore_index=True)

Step 4: Data Visualization

Now, let's visualize the data using Matplotlib to get a better understanding of the performance of these indices. We will plot a line graph with dates on the x-axis and index value on the y-axis.


# Plotting a line graph for S&P 500
plt.plot(sp500_df['date'], sp500_df['price'], label='S&P 500')

# Plotting a line graph for NASDAQ
plt.plot(nasdaq_df['date'], nasdaq_df['price'], label='NASDAQ')

# Plotting a line graph for FTSE
plt.plot(ftse_df['date'], ftse_df['price'], label='FTSE')

# Adding labels and legends
plt.xlabel('Date')
plt.ylabel('Index Value')
plt.title('Performance of Popular Indices')
plt.legend()
plt.show()

The output of the above code will be a line graph showing the performance of all three indices over time.

Step 5: Historical Data Analysis

To view historical trends, we will add a feature to our tracker that allows users to select a specific time period and view the performance of the indices during that period. We will use the datetime library to convert the dates into a more readable format and filter the data based on the selected time period.


# Converting dates to a more readable format
indices_df['date'] = pd.to_datetime(indices_df['date']).dt.date

# User input for time period
start_date = input('Enter start date (YYYY-MM-DD): ')
end_date = input('Enter end date (YYYY-MM-DD): ')

# Filtering data based on user input
filtered_df = indices_df[(indices_df['date'] >= start_date) & (indices_df['date'] <= end_date)]

# Plotting a line graph for filtered data
plt.plot(filtered_df['date'], filtered_df['price'], label='Indices')

# Adding labels and legends
plt.xlabel('Date')
plt.ylabel('Index Value')
plt.title('Performance of Popular Indices from {} to {}'.format(start_date, end_date))
plt.legend()
plt.show()

The output of the above code will be a line graph showing the performance of all three indices during the selected time period.
