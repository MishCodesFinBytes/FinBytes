What is Loan Amortization?

Loan amortization is the process of paying off a debt with regular payments, consisting of both principal and interest. An amortization schedule is a table that shows the breakdown of each payment, including the amount of principal and interest paid, remaining balance, and the total amount paid over the loan term. It is essential to understand the loan amortization schedule to make informed financial decisions, such as estimating the total cost of a loan or determining the impact of prepayments on the loan term.

Calculating Loan Amortization Schedules in Python:

Python offers various libraries and functions to calculate loan amortization schedules efficiently. Let's look at some of the ways we can use Python to automate this process.

Using the Numpy and Pandas Libraries:

Numpy and Pandas are two popular libraries used for data analysis and manipulation. We can use these libraries to create a loan amortization schedule in Python. First, we will define the loan details such as loan amount, interest rate, and loan term. Then, using the Numpy array function, we can create a table with the number of payments, payment amount, interest paid, and principal paid. Finally, we can use Pandas to display this data in a tabular format. Here's a code snippet to illustrate this:

# Importing necessary libraries
import numpy as np
import pandas as pd
import numpy_financial as npf  # Import numpy-financial for ipmt and ppmt

# Define loan details
loan_amount = 10000  # Total loan amount in currency
interest_rate = 0.05  # Annual interest rate (5%)
loan_term = 5  # Loan term in years

# Calculate the monthly interest rate
monthly_rate = interest_rate / 12  # Monthly interest rate (annual rate / 12)
nper = loan_term * 12  # Total number of payments (loan term in months)

# Calculate the monthly payment using the loan amortization formula
# The formula: M = P * [r(1 + r)^n] / [(1 + r)^n - 1]
# M = monthly payment, P = loan amount, r = monthly interest rate, n = number of periods
monthly_payment = loan_amount * (monthly_rate * (1 + monthly_rate) ** nper) / ((1 + monthly_rate) ** nper - 1)

# Generate the payment number sequence (from 1 to total number of months)
payment_number = np.arange(1, nper + 1)

# Calculate interest paid each month using numpy-financial's npf.ipmt function
interest_paid = npf.ipmt(monthly_rate, payment_number, nper, loan_amount)

# Calculate principal paid each month using numpy-financial's npf.ppmt function
principal_paid = npf.ppmt(monthly_rate, payment_number, nper, loan_amount)

# Create a DataFrame using Pandas to display the loan amortization table
df = pd.DataFrame({
    'Payment Number': payment_number,  # Payment number (month)
    'Payment Amount': np.repeat(monthly_payment, nper),  # Same monthly payment for all months
    'Interest Paid': interest_paid,  # Interest paid in each month
    'Principal Paid': principal_paid  # Principal paid in each month
})

# Display the loan amortization table without index
print(df.to_string(index=False))

Using Pure Python (No Libraries):

For smaller projects or educational purposes, you can calculate loan amortization directly in Python without any external libraries. By implementing the amortization formula manually, this approach offers full transparency into each calculation step. Here’s an outline of what this approach might look like:

def calculate_amortization_schedule(principal, annual_interest_rate, term_years):
    monthly_interest_rate = annual_interest_rate / 100 / 12
    num_payments = term_years * 12
    monthly_payment = principal * (monthly_interest_rate * (1 + monthly_interest_rate)**num_payments) / ((1 + monthly_interest_rate)**num_payments - 1)
    
    balance = principal
    schedule = []
    
    for i in range(1, num_payments + 1):
        interest = balance * monthly_interest_rate
        principal_payment = monthly_payment - interest
        balance -= principal_payment
        schedule.append((i, principal_payment, interest, balance if balance > 0 else 0))
    
    return schedule

This method allows for a highly customizable solution, although it requires more lines of code and knowledge of the amortization formula.

Using Financial Libraries like numpy_financial

Python has financial-specific libraries such as numpy_financial, which include functions designed for finance calculations like payment (pmt), present value (pv), and future value (fv). These functions can simplify calculations for amortization without needing a full library like loan_calculator.

import numpy_financial as npf

# Calculate monthly payment
principal = 10000
annual_interest_rate = 5
term_years = 5

monthly_payment = npf.pmt(annual_interest_rate / 100 / 12, term_years * 12, -principal)

print(monthly_payment)

This is particularly useful for developers already familiar with the finance-specific functions and who prefer to work within a streamlined environment.

Using SymPy for Symbolic Computation

SymPy is a symbolic computation library in Python that can solve mathematical equations, making it ideal for algebraic manipulation. For educational purposes or if you need exact symbolic results, you could use SymPy to symbolically solve for loan payments or balances over time.

from sympy import symbols, Eq, solve

# Define variables
P, r, n, M = symbols('P r n M')  # Principal, monthly interest rate, term in months, monthly payment
equation = Eq(M, P * r * (1 + r)**n / ((1 + r)**n - 1))

# Solve for M, the monthly payment
monthly_payment = solve(equation, M)[0]

While it’s not commonly used for amortization schedules, SymPy can offer insight into the formulas behind the calculations.

Using scipy.optimize to Solve for Payment Values

For more complex loans or unusual repayment schedules, SciPy’s optimize module can help solve for monthly payments based on constraints. This is especially useful in scenarios where payment amounts vary or custom repayment models are required.

For example, you could use scipy.optimize.fsolve to calculate the monthly payment that would reduce the loan balance to zero over time:

from scipy.optimize import fsolve

# Define a function for loan balance at term end given a monthly payment
def loan_balance(monthly_payment):
