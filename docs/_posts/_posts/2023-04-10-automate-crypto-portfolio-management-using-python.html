Cryptocurrencies have taken the financial world by storm, with their decentralized nature and potential for high returns attracting investors and traders alike. As the crypto market is highly volatile, it is essential for investors to keep track of their portfolio's performance in real-time. However, manually tracking and calculating gains and losses can be a tedious and time-consuming task. This is where Python comes in, with its versatile capabilities in automating financial tasks. In this blog post, we will explore how to build a Python script that tracks the value of a cryptocurrency portfolio in real-time, displaying gains, losses, and overall performance. We will also discuss how this concept can benefit the fintech industry and its potential for future applications.

Main Content:

To begin, we will need to have a basic understanding of the Python programming language and its libraries for data analysis and visualization. We will be using the following libraries for our script:

1. `requests`: for making HTTP requests to cryptocurrency APIs

2. `pandas`: for data manipulation and analysis

3. `matplotlib`: for data visualization

Step 1: Setting up the Environment

Before we dive into coding, we need to set up our development environment. We can use any text editor or IDE for this purpose; however, we recommend using Jupyter Notebook for its interactive and user-friendly interface. We also need to install the required libraries using the `pip` command in the terminal.

Step 2: Retrieving Cryptocurrency Data

To track the value of our cryptocurrency portfolio, we need to fetch the current prices of the cryptocurrencies we hold. We can do this by making HTTP requests to cryptocurrency APIs, such as CoinMarketCap, using the `requests` library. Let's take an example of tracking the value of Bitcoin and Ethereum.

import requests

# Bitcoin API endpoint

btc_endpoint = 'https://api.coinmarketcap.com/v1/ticker/bitcoin/'

# Ethereum API endpoint

eth_endpoint = 'https://api.coinmarketcap.com/v1/ticker/ethereum/'

# Get Bitcoin and Ethereum data
 btc_data = requests.get(btc_endpoint).json()
 eth_data = requests.get(eth_endpoint).json()

The `requests.get()` method returns a response object, which we convert to a JSON format using the `json()` method. We can now access the required data, such as the current price, using the key-value pairs in the JSON object.

Step 3: Creating a Portfolio

Next, we need to create a portfolio, which will hold the cryptocurrencies we want to track and their respective quantities. We can use a dictionary for this purpose, where the keys will be the cryptocurrency names, and the values will be their quantities.

# Create a portfolio
portfolio = {'BTC': 2, 'ETH': 5, 'ADA': 1000, 'SOL': 50}

Step 4: Calculating Portfolio Value

Now that we have the current prices of our cryptocurrencies and their quantities, we can calculate the total value of our portfolio. We will use the `pandas` library to create a DataFrame, which will hold the data.

import pandas as pd

# Create DataFrame

crypto_data = pd.DataFrame([btc_data[0], eth_data[0]])

# Add a 'Quantity' column to the DataFrame

crypto_data['Quantity'] = [portfolio['BTC'], portfolio['ETH']]

# Add a 'Value' column, which is the product of 'Price' and 'Quantity'
crypto_data['Value'] = crypto_data['price_usd'] * crypto_data['Quantity']

Step 5: Visualizing Portfolio Performance

We can now visualize the performance of our portfolio using the `matplotlib` library. Let's plot a bar chart to compare the current value of each cryptocurrency to its value at the time of purchase.

import matplotlib.pyplot as plt

# Plot a bar chart
 plt.bar(crypto_data['name'], crypto_data['Value'])
 plt.xlabel('Cryptocurrency')
 plt.ylabel('Value (USD)')
 plt.title('Portfolio Performance')
 plt.show()

This chart will show us the gains or losses we have incurred on each cryptocurrency. We can also add a line chart to see the overall performance of our portfolio over time.
