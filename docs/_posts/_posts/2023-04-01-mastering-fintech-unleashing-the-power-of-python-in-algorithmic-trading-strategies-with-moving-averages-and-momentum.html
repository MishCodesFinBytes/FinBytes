Moving Averages:

Moving averages (MA) is a widely used technical analysis tool in the financial industry. It is a mathematical calculation that smooths out price data by constantly updating the average price over a specific period. This helps in identifying the trend of a stock or security. Python has several libraries, such as Pandas and NumPy, that make it easy to calculate moving averages.

There are three types of moving averages - simple moving average (SMA), exponential moving average (EMA), and weighted moving average (WMA). Let's take a look at each one of them and how they can be implemented in Python.

1. Simple Moving Average (SMA):

SMA is the most basic form of moving average. It calculates the average price of a security over a specific period by adding the closing prices of each day and dividing it by the number of days in the period. The formula for SMA is as follows:

SMA = (Sum of Closing Prices over n-periods) / n

In Python, we can use the 'rolling' function from the Pandas library to calculate the SMA. Here's an example of calculating a 10-day SMA for Apple stock using Pandas and visualizing it using Matplotlib:


# Import necessary libraries
import pandas as pd
import matplotlib.pyplot as plt
import pandas_datareader as web

# Download Apple stock data from Yahoo Finance
df = web.DataReader('AAPL', data_source='yahoo', start='2020-01-01', end='2020-12-31')

# Calculate 10-day SMA
sma_10 = df['Close'].rolling(window=10).mean()

# Plotting the SMA
plt.figure(figsize=(10,5))
plt.plot(df['Close'], label='AAPL')
plt.plot(sma_10, label='10-day SMA')
plt.legend()
plt.show()

2. Exponential Moving Average (EMA):

EMA gives more weight to recent prices, making it more responsive to price changes than SMA. The formula for EMA is as follows:

EMA = (Closing Price * Multiplier) + (Previous EMA * (1-Multiplier))

Where the multiplier is calculated as (2 / (n + 1)), where n is the number of periods. In Python, we can use the 'ewm' function from the Pandas library to calculate EMA. Here's an example of calculating a 20-day EMA for Microsoft stock using Pandas and visualizing it using Matplotlib:


# Import necessary libraries
import pandas as pd
import matplotlib.pyplot as plt
import pandas_datareader as web

# Download Microsoft stock data from Yahoo Finance
df = web.DataReader('MSFT', data_source='yahoo', start='2020-01-01', end='2020-12-31')

# Calculate 20-day EMA
ema_20 = df['Close'].ewm(span=20, adjust=False).mean()

# Plotting the EMA
plt.figure(figsize=(10,5))
plt.plot(df['Close'], label='MSFT')
plt.plot(ema_20, label='20-day EMA')
plt.legend()
plt.show()

3. Weighted Moving Average (WMA):

WMA gives more weight to recent prices, similar to EMA, but it also assigns weights to each price based on the number of periods. The formula for WMA is as follows:

WMA = (P1 * n) + (P2 * (n-1)) + (P3 * (n-2)) + ... + (Pn * 1)

----------------------------------------------

(n + (n-1) + (n-2) + ... + 1)

Where P1 is the latest price, P2 is the price before P1, and so on. In Python, we can use the 'apply' function from the Pandas library to calculate WMA. Here's an example of calculating a 5-day WMA for Google stock using Pandas and visualizing it using Matplotlib:


# Import necessary libraries
import pandas as pd
import matplotlib.pyplot as plt
import pandas_datareader as web

# Download Google stock data from Yahoo Finance
df = web.DataReader('GOOGL', data_source='yahoo', start='2020-01-01', end='2020-12-31')

# Calculate 5-day WMA
def wma(data):
    weight = pd.Series(range(1, len(data) + 1), index=data.index)
    return data['Close'].rolling(len(data)).apply(lambda prices: prices.dot(weight) / weight.sum(), raw=True)

wma_5 = wma(df)

# Plotting the WMA
plt.figure(figsize=(10,5))
plt.plot(df['Close'], label='GOOGL')
plt.plot(wma_5, label='5-day WMA')
plt.legend()
plt.show()

Momentum Trading:

Momentum trading is a popular strategy that involves buying securities that are trending upwards and selling those that are trending downwards. It is based on the concept that a security that has been performing well in the past is likely to continue performing well in the future. Python has a powerful library called 'ta' that makes it easy to implement momentum trading strategies.

There are various indicators used in momentum trading, such as the Relative Strength Index (RSI), Moving Average Convergence Divergence (MACD), and Stochastic Oscillator. Let's take a look at how we can implement these indicators in Python.

1. Relative Strength Index (RSI):

RSI is a momentum indicator that measures the speed and change of price movements. It oscillates between 0 and 100 and is considered overbought when it is above 70 and oversold when it is below 30. In Python, we can use the 'rsi' function from the 'ta' library to calculate RSI. Here's an example of calculating a 14-day RSI for Tesla stock and visualizing it using Matplotlib:


# Import necessary libraries
import pandas as pd
import matplotlib.pyplot as plt
import pandas_datareader as web
import ta

# Download Tesla stock data from Yahoo Finance
df = web.DataReader('TSLA', data_source='yahoo', start='2020-01-01', end='2020-12-31')

# Calculate 14-day RSI
rsi_14 = ta.rsi(df['Close'], length=14)

# Plotting the RSI
plt.figure(figsize=(10,5))
plt.plot(df['Close'], label='TSLA')
plt.plot(rsi_14, label='14-day RSI')
plt.legend()
plt.show()

2. Moving Average Convergence Divergence (MACD):

MACD is a trend-following momentum indicator that calculates the difference between two moving averages of different lengths. In Python, we can use the 'macd' function from the 'ta' library to calculate MACD. Here's an example of calculating MACD for Amazon stock and visualizing it using Matplotlib:


# Import necessary libraries
import pandas as pd
import matplotlib.pyplot as plt
import pandas_datareader as web
import ta

# Download Amazon stock data from Yahoo Finance
df = web.DataReader('AMZN', data_source='yahoo', start='2020-01-01', end='2020-12-31')

# Calculate MACD
macd = ta.macd(df['Close'])

# Plotting the MACD
plt.figure(figsize=(10,5))
plt.plot(df['Close'], label='AMZN')
plt.plot(macd['MACD'], label='MACD')
plt.plot(macd['SIGNAL'], label='Signal')
plt.legend()
plt.show()

3. Stochastic Oscillator:

Stochastic Oscillator is a momentum indicator that compares the current closing price of a security to its price range over a specific period. It oscillates between 0 and 100 and is considered overbought when it is above 80 and oversold when it is below 20. In Python, we can use the 'stoch' function from the 'ta' library to calculate Stochastic Oscillator. Here's an example of calculating Stochastic Oscillator for Facebook stock and visualizing it using Matplotlib:


# Import necessary libraries
import pandas as pd
import matplotlib.pyplot as plt
import pandas_datareader as web
import ta

# Download Facebook stock data from Yahoo Finance
df = web.DataReader('FB', data_source='yahoo', start='2020-01-01', end='2020-12-31')

# Calculate Stochastic Oscillator
stoch = ta.stoch(df['High'], df['Low'], df['Close'])

# Plotting the Stochastic Oscillator
plt.figure(figsize=(10,5))
plt.plot(df['Close'], label='FB')
plt.plot(stoch['%K'], label='%K')
plt.plot(stoch['%D'], label='%D')
plt.legend()
plt.show()
