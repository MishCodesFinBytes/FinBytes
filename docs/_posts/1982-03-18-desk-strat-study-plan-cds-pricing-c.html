Projects

Project 1 — Bond Pricer & Yield Curve Bootstrap

Weeks 3–4 · ~18 hrs across all 3 languages

What you build:

  • Bootstrap a zero curve from par swap rates

  • Price a fixed-rate bond (clean/dirty price, accrued interest)

  • Compute modified duration and convexity

  • Benchmark: price 100,000 bonds — wall time + peak memory

C++ concepts covered: std::vector, const references, Newton-Raphson, class design (YieldCurve, Bond, CashFlow)

Project 2 — CDS Pricer with Sensitivities

Weeks 5–7 · ~28 hrs across all 3 languages

What you build:

  • Bootstrap piecewise-flat hazard rate curve from market CDS spreads

  • Price a CDS: fee leg, protection leg, upfront/par spread

  • Compute CS01, DV01, JTD via bump-and-reprice

  • Benchmark: bootstrap + reprice 10,000 curves with full Greeks

C++ concepts covered: template classes for term structure interpolation, QuantLib CreditDefaultSwap, profiling with gprof

Project 3 — Portfolio CDS Engine with Correlated Defaults

Weeks 8–10 · ~35 hrs across all 3 languages

What you build:

  • Portfolio of N single-name CDS positions

  • Gaussian copula Monte Carlo for correlated default simulation

  • Aggregate CS01, DV01, JTD across portfolio

  • Benchmark: 1M MC paths, 100-name portfolio — wall time + thread scaling

C++ concepts covered: std::thread/std::async, Cholesky decomposition (Eigen), AoS vs SoA data layout

Capstone — Full Credit Risk System

Weeks 11–12 · C++ core, Python reporting layer

What you build:

  • Load portfolio from CSV (names, notionals, maturities, market spreads)

  • Price all positions + compute full Greeks in C++

  • PnL explain: decompose daily P&L into CS01 carry, spread move, rate move, theta

  • Scenario shocks: parallel ±50bp, curve steepener/flattener

  • Python layer reads C++ output, generates terminal risk report

  • CLI: ./credit-risk portfolio.csv --date 2025-01-15

Key Resources

Things Most Candidates Miss

Trader communication — practice explaining each model in 90 seconds with no jargon. Record yourself. Desk strats spend more time talking to traders than writing code.

Market awareness — follow CDX IG and HY spreads weekly from week 1. Know the current credit cycle. Interviewers will ask.

The ISDA standard model — every bank prices CDS against it. Read the spec, implement it exactly, understand where your simplified model deviates.

Networking — after blog post 4 goes up, reach out to 2–3 quant finance people on LinkedIn per week. Your blog is the conversation starter.
