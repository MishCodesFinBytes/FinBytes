Let’s upgrade our single-file chat app to log cache hits/misses, message publishing, and Pub/Sub events, so you can see issues live in the console for demonstration and troubleshooting.

Updated Single-File Chat + Logging Demo – chat_cache_demo_logs.py

import redis
import threading
import time
from flask import Flask, request, jsonify

# ------------------------------
# Redis Connection
# ------------------------------
r = redis.Redis(host='localhost', port=6379, db=0, decode_responses=True)

# ------------------------------
# Chat and Cache with Logging
# ------------------------------
class ChatCacheDemo:
    """Chat app with Redis caching + Pub/Sub + detailed logging"""

    MESSAGE_LIST_KEY = "chat_messages"  # Last N messages
    CACHE_TTL = 300  # 5 minutes

    @staticmethod
    def log(msg):
        print(f"[LOG] {time.strftime('%H:%M:%S')} - {msg}")

    @staticmethod
    def start_listener():
        pubsub = r.pubsub()
        pubsub.subscribe("global_chat")
        ChatCacheDemo.log("Chat listener started.")
        for message in pubsub.listen():
            if message['type'] == 'message':
                ChatCacheDemo.log(f"[PUB/SUB RECEIVED] {message['data']}")

    @staticmethod
    def send_message(message):
        try:
            # Publish to live subscribers
            r.publish("global_chat", message)
            ChatCacheDemo.log(f"[PUB/SUB SENT] {message}")

            # Add to cache
            r.lpush(ChatCacheDemo.MESSAGE_LIST_KEY, message)
            r.ltrim(ChatCacheDemo.MESSAGE_LIST_KEY, 0, 9)  # Keep last 10
            r.expire(ChatCacheDemo.MESSAGE_LIST_KEY, ChatCacheDemo.CACHE_TTL)
            ChatCacheDemo.log(f"[CACHE] Message added, TTL={ChatCacheDemo.CACHE_TTL}s")

        except redis.exceptions.ConnectionError:
            ChatCacheDemo.log("[ERROR] Redis connection failed! Message not sent.")

    @staticmethod
    def get_cached_messages():
        try:
            messages = r.lrange(ChatCacheDemo.MESSAGE_LIST_KEY, 0, -1)
            if messages:
                ChatCacheDemo.log(f"[CACHE HIT] Retrieved {len(messages)} messages")
            else:
                ChatCacheDemo.log("[CACHE MISS] No messages in cache")
            return messages
        except redis.exceptions.ConnectionError:
            ChatCacheDemo.log("[ERROR] Redis connection failed! Cannot retrieve cache")
            return []

# ------------------------------
# Start listener thread
# ------------------------------
threading.Thread(target=ChatCacheDemo.start_listener, daemon=True).start()

# ------------------------------
# Flask App
# ------------------------------
app = Flask(__name__)

@app.route("/chat", methods=["POST"])
def send_chat():
    data = request.json
    message = data.get("message")
    if message:
        ChatCacheDemo.send_message(message)
        return jsonify({"status": "sent"})
    return jsonify({"status": "failed", "error": "No message provided"}), 400

@app.route("/history")
def chat_history():
    messages = ChatCacheDemo.get_cached_messages()
    return jsonify({"messages": messages})

@app.route("/ping")
def ping():
    return "Server running!"

# ------------------------------
# Run App
# ------------------------------
if __name__ == "__main__":
    ChatCacheDemo.log("Starting Chat Cache Demo App with logging...")
    app.run(debug=True)

Step 1: Run Instructions (Windows)

  1. Start Redis:

redis-server.exe

  1. Run Python app:

python chat_cache_demo_logs.py

  • Flask server starts at http://127.0.0.1:5000/

  • Console shows logging for all events (cache hits/misses, Pub/Sub messages)

Step 2: Test & Demonstrate Issues

  1. Cache Miss

    • Open /history before sending any messages

    • Console logs: [CACHE MISS] No messages in cache

  2. Send Messages

POST /chat {"message":"Hello World"}

  • Console logs:

[LOG] HH:MM:SS - [PUB/SUB SENT] Hello World
[LOG] HH:MM:SS - [CACHE] Message added, TTL=300s
[LOG] HH:MM:SS - [PUB/SUB RECEIVED] Hello World

  1. Cache Hit

    • Open /history again

    • Logs: [CACHE HIT] Retrieved 1 messages

  2. Cache Expiration

    • Wait >5 minutes → /history shows [CACHE MISS]

  3. Redis Down

    • Stop Redis → send message → logs: [ERROR] Redis connection failed! Message not sent.

  4. Multi-Listener / Race Conditions

    • Run another Python instance of the same app → messages are received on all listeners

    • Console shows [PUB/SUB RECEIVED] on each listener
