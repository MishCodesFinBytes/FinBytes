The STL container zoo

C++ ships with 14+ container types. Python has 4 (list, dict, set, tuple). Knowing which C++ container to reach for — and why — is one of the first real C++ skills.

The decision in one sentence

Default to std::vector. Switch to something else only when you have a measured reason.

Quick reference

  • std::vector<T> — dynamic array. O(1) random access, O(1) amortised push_back. Use for almost everything.
  • std::array<T,N> — fixed-size array, stack allocated. Use when size is compile-time constant.
  • std::deque<T> — double-ended queue. O(1) push_front and push_back. Use for sliding windows.
  • std::list<T> — doubly linked list. O(1) insert/erase anywhere. Rarely worth it — poor cache performance.
  • std::map<K,V> — sorted by key, O(log n). Use for order books (need sorted iteration).
  • std::unordered_map<K,V> — hash map, O(1) average. Use for fast lookup by key.
  • std::set<T> / std::unordered_set<T> — unique elements, same tradeoffs as map.
  • std::span<T> — non-owning view into contiguous data. Use for function parameters.

Python container equivalents

from collections import deque, defaultdict, OrderedDict
import heapq

# list → std::vector (dynamic array)
prices = [100.0, 101.5, 99.8]
prices.append(102.0)
prices.insert(0, 99.0)  # O(n) — expensive

# dict → std::unordered_map (hash map)
positions = {"AAPL": 100.0, "MSFT": 200.0}
positions["GOOGL"] = 50.0

# set → std::unordered_set
symbols = {"AAPL", "MSFT", "GOOGL"}
symbols.add("TSLA")

# deque → std::deque
window = deque(maxlen=20)  # circular buffer
for p in prices:
    window.append(p)

# heapq → std::priority_queue
heap = []
heapq.heappush(heap, (100.0, "AAPL"))
best = heapq.heappop(heap)

# OrderedDict → std::map (sorted)
book = OrderedDict(sorted({"100.5": 1000, "99.5": 500}.items()))

C# container equivalents

using System.Collections.Generic;

// List<T> → std::vector
var prices = new List<double> { 100.0, 101.5, 99.8 };
prices.Add(102.0);

// T[] → std::array
double[] fixedPrices = new double[256];

// Dictionary<K,V> → std::unordered_map
var positions = new Dictionary<string, double>();
positions["AAPL"] = 100.0;

// SortedDictionary<K,V> → std::map
var orderBook = new SortedDictionary<double, int>();
orderBook[100.5] = 1000;

// HashSet<T> → std::unordered_set
var symbols = new HashSet<string> { "AAPL", "MSFT" };

// Queue<T> → std::queue
var tickQueue = new Queue<double>();
tickQueue.Enqueue(100.0);
var next = tickQueue.Dequeue();

// PriorityQueue<T,P> → std::priority_queue
var pq = new PriorityQueue<string, double>();
pq.Enqueue("AAPL", -100.5);  // negate for max-heap
var (sym, pri) = (pq.Peek(), 0.0);

C++ STL containers in practice

#include <vector>
#include <array>
#include <deque>
#include <map>
#include <unordered_map>
#include <set>
#include <queue>
#include <span>
#include <string>
#include <iostream>

// vector — default choice
std::vector<double> prices = {100.0, 101.5, 99.8};
prices.push_back(102.0);
prices.reserve(1000);  // pre-allocate

// array — compile-time size, stack allocated
std::array<double, 256> buffer{};
buffer.fill(0.0);

// span — non-owning view, function parameter
double mean(std::span<const double> v) {
    return std::reduce(v.begin(), v.end()) / v.size();
}
// Works with vector, array, C-array — no copies
double m = mean(prices);

// deque — O(1) both ends
std::deque<double> window;
for (double p : prices) {
    window.push_back(p);
    if (window.size() > 20) window.pop_front();
}

// map — sorted by key (use for order book)
std::map<double, int, std::greater<double>> bids;  // descending price
bids[100.5] = 1000;
bids[99.5]  = 500;
auto best_bid = bids.begin();  // highest price first

// unordered_map — O(1) lookup
std::unordered_map<std::string, double> positions;
positions.reserve(100);
positions["AAPL"] = 100.0;
auto it = positions.find("AAPL");
if (it != positions.end()) std::cout << it->second << "
";

// priority_queue — max heap by default
std::priority_queue<std::pair<double,std::string>> pq;
pq.push({100.5, "AAPL"});
auto [price, sym] = pq.top();

// set — unique sorted elements
std::set<std::string> liquid_symbols = {"AAPL", "MSFT", "GOOGL"};
bool is_liquid = liquid_symbols.count("AAPL") > 0;
💡 Use std::span<const T> for function parameters that just need to read a sequence — it accepts vector, array, raw pointer+size, and std::array without any copies or template instantiation bloat. It's the C++ equivalent of Python's duck typing for sequences.
