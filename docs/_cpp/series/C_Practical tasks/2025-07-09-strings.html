Strings are harder in C++

Python and C# treat string manipulation as a first-class operation. C++ std::string is fine for most tasks but lacks many conveniences. C++20 added std::format and std::string_view dramatically improved performance for read-only string operations.

string_view

A non-owning view into a string — just a pointer and a length. No allocation. Pass string literals, substrings, or existing strings without copying. The first C++ string feature that made me smile.

std::format (C++20)

Finally, Python-style format strings in C++. std::format("Price: {:.2f}", price) instead of std::ostringstream gymnastics.

// Formatting — C++20
double price  = 100.25;

// Split on delimiter
std::vector<std::string> split(std::string_view s, char delim) {
    std::vector<std::string> result;
    std::string part;
    for (char c : s) {
        if (c == delim) { result.push_back(part); part.clear(); }
        else part += c;
    }
    result.push_back(part);
    return result;
}

// Trim whitespace
std::string trim(std::string_view s) {
    auto start = s.find_first_not_of(" \t\r\n");
    auto end   = s.find_last_not_of(" \t\r\n");
    if (start == std::string_view::npos) return "";
    return std::string(s.substr(start, end - start + 1));
}

// string_view — zero-copy substring
void parse_csv_line(std::string_view line) {
    std::size_t pos = 0, next;
    while ((next = line.find(',', pos)) != std::string_view::npos) {
        std::string_view field = line.substr(pos, next - pos);
        // process field — no allocation
        pos = next + 1;
    }
    std::string_view last = line.substr(pos);
}

// Regex
#include <regex>
std::string line = "AAPL,100,150.25,BUY";
std::regex pattern(R"((\w+),(\d+),([\d.]+))");
std::smatch m;
if (std::regex_search(line, m, pattern)) {
    auto sym = m[1].str();
    auto qty = m[2].str();
    auto px  = m[3].str();
}
💡 std::string_view is the single biggest quality-of-life improvement for C++ string handling. Pass it instead of const std::string& whenever you don't need to own the string — it accepts string literals, substrings, and std::string with no copying.
