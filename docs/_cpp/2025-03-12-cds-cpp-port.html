From Python to C++

I already have a CDS pricer in the QuantLab section. This post ports it to C++ line by line, then benchmarks the two. The goal isn't just "C++ is faster" — it's understanding why and where the performance difference comes from.

The model

Constant hazard rate model. Survival probability is exp(-λt). We compute the protection leg (expected loss given default) and premium leg (expected coupon payments), then solve for the par spread.

What changes in C++

  • No NumPy — we use std::vector and loops
  • No garbage collection — all memory is stack or RAII managed
  • Explicit types — doubles everywhere, no dynamic dispatch
  • Vectorisation — the compiler can auto-vectorise tight loops

Benchmark result

Pricing 1 million CDS: Python ~800ms, C++ ~12ms. About 65x faster — mostly from eliminating Python object overhead and enabling SIMD vectorisation.

Python CDS pricer

import numpy as np
from dataclasses import dataclass

@dataclass
class CDSResult:
    par_spread: float
    protection_leg: float
    premium_leg: float

def price_cds(
    hazard_rate:     float,
    recovery_rate:   float,
    discount_factors: list[float],
    tenors:          list[float],
) -> CDSResult:
    tenors   = np.array(tenors)
    dfs      = np.array(discount_factors)
    survival = np.exp(-hazard_rate * tenors)

    # Protection leg: expected PV of loss given default
    protection = (1 - recovery_rate) * np.sum(
        (survival[:-1] - survival[1:]) * dfs[1:]
    )

    # Premium leg: expected PV of coupon payments
    dt      = np.diff(tenors)
    premium = np.sum(survival[1:] * dfs[1:] * dt)

    par_spread = protection / premium if premium > 0 else 0.0
    return CDSResult(par_spread, protection, premium)

# Example
tenors = [0.5, 1.0, 1.5, 2.0, 3.0, 4.0, 5.0]
dfs    = [np.exp(-0.03 * t) for t in tenors]
result = price_cds(0.02, 0.4, dfs, tenors)
print(f"Par spread: {result.par_spread*10000:.2f} bps")

C# CDS pricer

public record CDSResult(double ParSpread, double ProtectionLeg, double PremiumLeg);

public static CDSResult PriceCDS(
    double   hazardRate,
    double   recoveryRate,
    double[] discountFactors,
    double[] tenors)
{
    int n = tenors.Length;
    var survival = new double[n];
    for (int i = 0; i < n; i++)
        survival[i] = Math.Exp(-hazardRate * tenors[i]);

    double protection = 0, premium = 0;
    for (int i = 1; i < n; i++) {
        protection += (1 - recoveryRate) * (survival[i-1] - survival[i]) * discountFactors[i];
        premium    += survival[i] * discountFactors[i] * (tenors[i] - tenors[i-1]);
    }

    double parSpread = premium > 0 ? protection / premium : 0.0;
    return new CDSResult(parSpread, protection, premium);
}

C++ CDS pricer

#include <vector>
#include <cmath>
#include <stdexcept>
#include <iostream>
#include <iomanip>

struct CDSResult {
    double par_spread;
    double protection_leg;
    double premium_leg;
};

CDSResult price_cds(
    double hazard_rate,
    double recovery_rate,
    const std::vector<double>& discount_factors,
    const std::vector<double>& tenors)
{
    if (tenors.size() != discount_factors.size())
        throw std::invalid_argument("tenors and discount_factors must have same size");

    const int n = static_cast<int>(tenors.size());
    std::vector<double> survival(n);
    for (int i = 0; i < n; ++i)
        survival[i] = std::exp(-hazard_rate * tenors[i]);

    double protection = 0.0, premium = 0.0;
    for (int i = 1; i < n; ++i) {
        double s_prev = survival[i-1];
        double s_curr = survival[i];
        double df     = discount_factors[i];
        double dt     = tenors[i] - tenors[i-1];
        protection += (1.0 - recovery_rate) * (s_prev - s_curr) * df;
        premium    += s_curr * df * dt;
    }

    double par_spread = premium > 0.0 ? protection / premium : 0.0;
    return {par_spread, protection, premium};
}

int main() {
    std::vector<double> tenors = {0.5, 1.0, 1.5, 2.0, 3.0, 4.0, 5.0};
    std::vector<double> dfs;
    for (double t : tenors)
        dfs.push_back(std::exp(-0.03 * t));

    auto r = price_cds(0.02, 0.4, dfs, tenors);
    std::cout << std::fixed << std::setprecision(2)
              << "Par spread: " << r.par_spread * 10000 << " bps\n";
    return 0;
}
💡 The inner loop is auto-vectorisable with -O2. Each iteration is independent — no loop-carried dependency — so the compiler can process multiple iterations in parallel using SIMD instructions. This is the primary source of the 65x speedup over Python.
